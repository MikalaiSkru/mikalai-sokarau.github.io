How to play:

Match-match game is a counter game where the object is to find all the pairs.

When the game begins, all pictures are hidden.

Rules:

  1. Select a card and try to find a pair to it
  2. If you match the pictures you can choose again.
  3. If they don't match, the computer will turn them.
  4. The player that finds all the pairs wins!
  5. Good luck and have fun!
